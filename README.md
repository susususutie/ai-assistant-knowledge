# AI 助手知识库

这是一个用于指导 AI 工具进行开发工作的知识库。

## 内容结构

- [用户偏好](./user-preferences.md) - 用户的开发偏好和项目信息
- [项目索引](./project-index.md) - 各个项目的详细信息
- [最佳实践](./best-practices.md) - 代码审查中总结的最佳实践
- [禁止事项](./forbidden-actions.md) - 需要避免的行为

## 使用说明

任何帮助用户的 AI 工具在开始工作前应先阅读此知识库，确保：
1. 了解用户的编码风格偏好
2. 知道哪些项目是项目代码，哪些是学习示例
3. 遵循最佳实践
4. 避免执行禁止事项

## 更新维护

当 AI 发现新的有价值经验时，应更新此知识库。
