# 项目索引

## antd-dual-theme-manager

- **描述**: Ant Design 双主题管理器
- **技术栈**: React, Ant Design, TypeScript
- **用途**: 管理 Ant Design 组件的亮色/暗色主题切换
- **维护建议**: 保持 TypeScript 类型完整，移除 console.log

## antd-dynamic-theme

- **描述**: Ant Design 动态主题
- **技术栈**: React, Ant Design
- **用途**: 运行时动态切换主题色

## bash-bucket

- **描述**: Bash 脚本工具集
- **技术栈**: Bash
- **用途**: 常用 Bash 脚本，特别是 npm 镜像测速
- **维护建议**: 添加 set -u 和 set -o pipefail，使用 local 声明局部变量

## docker-setup

- **描述**: Docker 环境配置
- **技术栈**: Docker, JavaScript
- **用途**: Docker 相关工具和配置

## monorepo-setup

- **描述**: Monorepo 脚手架
- **技术栈**: Node.js
- **用途**: Monorepo 项目初始化配置

## note

- **描述**: 笔记文档
- **技术栈**: VitePress
- **用途**: 个人知识库文档
- **维护建议**: 修复文件句柄泄漏，使用 try-finally

## react-responsive-scale

- **描述**: React 响应式缩放组件
- **技术栈**: React, TypeScript
- **用途**: 响应式缩放 UI 组件
- **维护建议**: 移除 console.log，优化 useEffect/useMemo 使用
