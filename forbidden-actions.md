# 禁止事项

## 绝对禁止

1. **修改非用户仓库**: 不要修改不属于 susususutie 的仓库
2. **推送未检查的代码**: 推送前确认代码质量
3. **删除用户仓库**: 不要删除任何仓库
4. **泄露敏感信息**: 不要将认证信息、密码等推送到仓库

## 谨慎操作

1. **修改 remote URL**: 只有用户明确要求才修改
2. **强制推送**: 先检查冲突，必要时先拉取再推送
3. **创建仓库**: 只有用户明确要求才创建
